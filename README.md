# 🧑‍💻 Sobre mim

Sou Desenvolvedor **Full Stack .NET**, focado na construção de aplicações web modernas, performáticas e escaláveis.

Trabalho principalmente com **ASP.NET Core, C#, .NET, Entity Framework e SQL Server**, desenvolvendo APIs REST e aplicando boas práticas como **Clean Code e arquitetura em camadas**.

Estou aberto a oportunidades como Desenvolvedor .NET / Full Stack, mantendo também interesse em outras tecnologias do ecossistema web.

Atualmente desenvolvo projetos próprios para consolidar minhas habilidades e aplicar conceitos de desenvolvimento full stack na prática. Entre eles está a plataforma TreineMais, um sistema web para academias e personal trainers criarem treinos personalizados, gerenciarem alunos e acompanharem a evolução dos exercícios.

O projeto utiliza ASP.NET Core, Identity para autenticação e controle de acesso, EF Core e SQL Server, além de deploy utilizando Docker e Nginx.

Também venho desenvolvendo outros projetos, como um sistema de agendamento de serviços e um sistema de controle de estoque para restaurantes, reforçando experiência prática na construção de aplicações web completas.
